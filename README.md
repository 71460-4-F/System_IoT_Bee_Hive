# System_IoT_Bee_Hive
[![Visits Badge](https://badges.pufler.dev/visits/71460-4-F/System_IoT_Bee_Hive)](https://badges.pufler.dev)

## A prototype of a simple system for a possible use in IoT for monitoring hives.

This work had the theme focused on beekeeping. In addition to bringing the importance of beekeeping, as well as the work carried out by bees in society and in the environment, the main objective of the project is a proposal for a solution or minimization of the global problem of damage and theft in bee hives based on IoT and with low cost.
